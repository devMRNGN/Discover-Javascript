# Devtools -> NETWORK

* F12 atalho para abrir o devtools -> chrome <br>

* Network -> para ver as atividades da rede, os requests e as responses <br>

Enviamos as informações via HTTP para o servidor em forma de mensagem e ele nos responde nos enviando de volta a pagina, ou arquivos e etc <br>

* General, Response Headers, Request Headers <br>

* Cada HTTP que o arquivo encontra, ele cria um novo pedido e uma nova resposta e assim por diante
